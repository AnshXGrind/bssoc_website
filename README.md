# 🚀 BSSOC (Build. Ship. Scale. Open Code)

**An elite open-source program focused on real-world product building.**

## 🌟 Overview

The official platform for BSSOC. The application is built with a moody, cinematic dark theme focusing on user experience, smooth parallax animations, and high performance.

## 🛠️ Tech Stack

- **Framework:** [Next.js 15+ (App Router)](https://nextjs.org/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/) (Scroll progress, parallax, loading state sequences)
- **Language:** [TypeScript](https://www.typescriptlang.org/)

## 📦 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm, pnpm, yarn, or bun

### Installation & Run

1. Clone the repository:
   `ash
   git clone https://github.com/AnshXGrind/bssoc_website.git
   `
2. Navigate into the project:
   `ash
   cd bssoc_website
   `
3. Install dependencies and start the development server:
   `ash
   npm install
   npm run dev
   `

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🎨 UI/UX Features

- **Cinematic Loading Screen:** A smooth, word-by-word roll-up introduction sequence.
- **Parallax Mountain Scene:** Dynamic 2D SVG mountains that respond seamlessly to user scroll.
- **Monochrome & Slate Aesthetics:** Strict adherence to a sleek #0f172a workspace.
- **Magnetic Components & Smooth Scroll:** Premium interaction design using custom cursors and magnetic button constraints.

## 🚀 Deployment

This project is optimized for deployment on [Vercel](https://vercel.com/) or Netlify. The current production is compiled successfully using Next.js Turbopack.
